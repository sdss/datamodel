## Basic Information
Targets for robostrategy run. Each entry corresponds to a row of the carton_to_target table of targetdb. This file only contains a subset of the cartons, and gathers other information for use by robostrategy. Produced by robostrategy's rs_targets_extract.

### Naming Convention
$ROBOSTRATEGY_DATA/allocations/[PLAN]/rsTargets-[PLAN]-[OBSERVATORY].fits

### Releases
WORK

### Enviroments
ROBOSTRATEGY_DATA

### Approximate Size
16 GB

### File Type
FITS

### Generated by Product
