## Basic Information
The spField epoch files contain the epoch combined spectra for all exposures of a given field epoch (for an MJD or range of MJDs). This is the updated SDSSV version of the spPlate files and analogous to the spField files

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/[FIELD]/epoch/spField-[FIELD]-[MJD].fits

### Releases
IPL3, DR19, IPL4, DR20

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
64 MB

### File Type
FITS

### Generated by Product
idlspec2d

### Is a VAC
False

### Data Level
2.2.3
