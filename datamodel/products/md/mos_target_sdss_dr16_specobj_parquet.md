## Basic Information
This table contains the list of all SDSS optical spectra for a given data release, with associated parameters from the 2D and 1D pipelines for each.

The table contains both the BOSS and SDSS spectrograph data. The database representation is derived from the flat file information described here: <a href="<a href="https://data.sdss.org/datamodel/files/SPECTRO_REDUX/specObj.html">https://data.sdss.org/datamodel/files/SPECTRO_REDUX/specObj.html</a>">https://data.sdss.org/datamodel/files/SPECTRO_REDUX/specObj.html">https://data.sdss.org/datamodel/files/SPECTRO_REDUX/specObj.html</a></a> Note: the order of the columns in this documentation may not match the order of the columns in the database table.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_sdss_dr16_specobj.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.
