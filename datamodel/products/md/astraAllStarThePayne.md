## Basic Information
Results from the astraAllStarThePayne astra pipeline for each star. 

### Naming Convention
$MWM_ASTRA/[ASTRA_VERSION]/[RUN2D]-[APRED]/results/summary/astraAllStar-ThePayne-[ASTRA_VERSION]-[RUN2D]-[APRED].fits

### Releases
IPL1, IPL2, WORK

### Enviroments
MWM_ASTRA

### Approximate Size
99 MB

### File Type
FITS

### Generated by Product
astra

### Is a VAC
False
