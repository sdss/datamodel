## Basic Information
This table stores magnitude information for a target. Optical magnitudes that are not selected from SDSS photometry have been converted to the SDSS system.

Entries in this revised_magnitude table only exist if there was an issue with the values in the original magnitude table. Therefore: if a carton_to_target_pk exists in this table, the values in this table should be used instead.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_revised_magnitude.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.
