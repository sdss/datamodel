## Basic Information
Contains a summary of the Spectro-1D spZline output for all spectra in a field-mjd with line measurements

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/spectra/daily/full/[FIELDGRP]/[FIELD]/[MJD]/spAllLine-[FIELD]-[MJD].fits.gz

### Releases
IPL3, DR19, IPL4, DR20

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
391 KB

### File Type
FITS

### Generated by Product
idlspec2d

### Is a VAC
False

### Data Level
2.2.3

### HDUS List for release DR20
  - [HDU0: PRIMARY](#hdu0-primary)
  - [HDU1: SPLINE](#hdu1-spline)
