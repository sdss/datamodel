## Basic Information
Coadded guider images appropriate for a given exposure so that we have a time-integrated PSF.

### Naming Convention
cogimg-[mjd]-[exposure].fits.gz where [mjd] is the MJD of observation, and [exposure] is the (zero-padded) 8-digit exposure number.

### Releases
DR13, DR14, DR15, DR16, DR17

### Enviroments
MANGA_SPECTRO_REDUX

### Approximate Size
2 MB

### File Type
FITS

### Generated by Product
mangadrp: coaddgimg.pro

### Is a VAC
False

### Data Level
1.3.1
