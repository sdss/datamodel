## Basic Information
Gaia DR2 Source Table. See https://gea.esac.esa.int/archive/documentation/GDR2/Gaia_archive/chap_datamodel/sec_dm_main_tables/ssec_dm_gaia_source.html for complete details.

### Naming Convention
$MOS_TARGET/[V_TARG]/mos_gaia_dr2_source-[NUM].fits, where V_TARG=1.0.1 for DR18; and NUM = 1..1 to partition the file into parts

### Releases
DR18, DR19

### Enviroments
MOS_TARGET

### Approximate Size
999 MB

### File Type
FITS

### Generated by Product
sdss5db> targetdb, operations database server

### Is a VAC
