## Basic Information
apPlate files combine multiple exposures in a visit, which include
exposures at different dither positions. As a result, they represent
an adequately sampled version of the spectrum. The multiple exposures
are "flux-matched" to each other, and then roughly flux calibrated using
previously determined apFlux calibrations. Native wavelength sampling
is still preserved. These files still have all 300 spectra from a given
plate in a single file.

### Naming Convention
<code>apPlate-[abc]-[0-9]{4}-[0-9]{5}\.fits</code>, where
<code>[abc]</code> is the chip id,
<code>[0-9]{4}</code> is the 4-digit plate number
and <code>[0-9]{5}</code> is the 5-digit MJD number.
