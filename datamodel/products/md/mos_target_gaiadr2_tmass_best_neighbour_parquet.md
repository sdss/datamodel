## Basic Information
The Gaia DR2 vs. 2MASS PSC crossmatch provided by the Gaia collaboration.

See <a href="https://gea.esac.esa.int/archive/documentation/GDR2/Catalogue_consolidation/chap_cu9val_cu9val/ssec_cu9xma/sssec_cu9xma_extcat.html">https://gea.esac.esa.int/archive/documentation/GDR2/Catalogue_consolidation/chap_cu9val_cu9val/ssec_cu9xma/sssec_cu9xma_extcat.html</a> for complete details.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_gaiadr2_tmass_best_neighbour.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

## Example DATAFRAME List

### Columns


#### tmass_oid

- Column Name: tmass_oid
- Description: Additional numeric unique source identifier of 2MASS, increasing with declination
- Unit: N/A
- Data Type: Int64


#### number_of_neighbours

- Column Name: number_of_neighbours
- Description: Number of sources in the 2MASS Catalogue which match the Gaia source within position errors
- Unit: N/A
- Data Type: Int32


#### number_of_mates

- Column Name: number_of_mates
- Description: Number of other Gaia sources that have as best-neighbour the same 2MASS source.
- Unit: N/A
- Data Type: Int32


#### best_neighbour_multiplicity

- Column Name: best_neighbour_multiplicity
- Description: Number of neighbours with same probability as best neighbour
- Unit: N/A
- Data Type: Int32


#### source_id

- Column Name: source_id
- Description: Unique Gaia DR2 source identifier
- Unit: N/A
- Data Type: Int64


#### original_ext_source_id

- Column Name: original_ext_source_id
- Description: The unique source identifier in the original 2MASS catalogue
- Unit: N/A
- Data Type: String


#### angular_distance

- Column Name: angular_distance
- Description: Angular distance between the two sources
- Unit: arcsec
- Data Type: Float64


#### gaia_astrometric_params

- Column Name: gaia_astrometric_params
- Description: Number of Gaia astrometric params used
- Unit: N/A
- Data Type: Int32


#### tmass_pts_key

- Column Name: tmass_pts_key
- Description: key for crossmatch
- Unit: N/A
- Data Type: Int32
