## Basic Information
Summary file of all visits of stars included in MaNGA Stellar Library.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_mastar_goodvisits.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

### Releases
DR20

### Enviroments
MOS_TARGET

### Approximate Size
2 MB

### File Type
PARQUET

### Generated by Product
sdss5db targeting database.

### Is a VAC
False

### Data Level
2.3.3

### DATAFRAME List for release DR20
- Primary data frame.

## Example DATAFRAME List

### Columns


#### drpver

- Column Name: drpver
- Description: Version of mangadrp.
- Unit: N/A
- Data Type: String


#### mprocver

- Column Name: mprocver
- Description: Version of mastarproc.
- Unit: N/A
- Data Type: String


#### mangaid

- Column Name: mangaid
- Description: MaNGA-ID for the object.
- Unit: N/A
- Data Type: String


#### plate

- Column Name: plate
- Description: Plate ID.
- Unit: N/A
- Data Type: Int32


#### ifudesign

- Column Name: ifudesign
- Description: IFUDESIGN for the fiber bundle.
- Unit: N/A
- Data Type: String


#### mjd

- Column Name: mjd
- Description: Modified Julian Date for this visit.
- Unit: N/A
- Data Type: Int32


#### ifura

- Column Name: ifura
- Description: Right Ascension of the center of the IFU.
- Unit: deg
- Data Type: Float64


#### ifudec

- Column Name: ifudec
- Description: Declination of the center of the IFU.
- Unit: deg
- Data Type: Float64


#### ra

- Column Name: ra
- Description: Right Ascension for this object at the time given by the EPOCH column (Equinox J2000).
- Unit: deg
- Data Type: Float64


#### dec

- Column Name: dec
- Description: Declination for this object at the time given by the EPOCH column (Equinox J2000).
- Unit: deg
- Data Type: Float64


#### epoch

- Column Name: epoch
- Description: Epoch of the astrometry (which is approximate for some catalogs).
- Unit: N/A
- Data Type: Float32


#### psfmag_1

- Column Name: psfmag_1
- Description: PSF magnitude in the first band with the filter correspondence depending on PHOTOCAT.
- Unit: mag
- Data Type: Float32


#### psfmag_2

- Column Name: psfmag_2
- Description: PSF magnitude in the second band with the filter correspondence depending on PHOTOCAT.
- Unit: mag
- Data Type: Float32


#### psfmag_3

- Column Name: psfmag_3
- Description: PSF magnitude in the third band with the filter correspondence depending on PHOTOCAT.
- Unit: mag
- Data Type: Float32


#### psfmag_4

- Column Name: psfmag_4
- Description: PSF magnitude in the fourth band with the filter correspondence depending on PHOTOCAT.
- Unit: mag
- Data Type: Float32


#### psfmag_5

- Column Name: psfmag_5
- Description: PSF magnitude in the fifth band with the filter correspondence depending on PHOTOCAT.
- Unit: mag
- Data Type: Float32


#### mngtarg2

- Column Name: mngtarg2
- Description: MANGA_TARGET2 targeting bitmask.
- Unit: N/A
- Data Type: Int32


#### exptime

- Column Name: exptime
- Description: Median exposure time for all exposures on this visit.
- Unit: sec
- Data Type: Float32


#### nexp_visit

- Column Name: nexp_visit
- Description: Total number of exposures during this visit. This column was named 'NEXP' in DR15/16.
- Unit: N/A
- Data Type: Int16


#### nvelgood

- Column Name: nvelgood
- Description: Number of exposures (from all visits of this PLATE-IFUDESIGN) with good velocity measurements.
- Unit: N/A
- Data Type: Int16


#### heliov

- Column Name: heliov
- Description: Median heliocentric velocity of all exposures on all visits that yield good velocity measurements. This is used to shift the spectra to the rest frame.
- Unit: km/s
- Data Type: Float32


#### verr

- Column Name: verr
- Description: 1-sigma uncertainty of the heliocentric velocity
- Unit: km/s
- Data Type: Float32


#### v_errcode

- Column Name: v_errcode
- Description: Error code for HELIOV. Zero is good, nonzero is bad.
- Unit: N/A
- Data Type: Int16


#### heliov_visit

- Column Name: heliov_visit
- Description: Median heliocentric velocity of good exposures from only this visit, rather than from all visits.
- Unit: km/s
- Data Type: Float32


#### verr_visit

- Column Name: verr_visit
- Description: 1-sigma uncertainty of HELIOV_VISIT.
- Unit: km/s
- Data Type: Float32


#### v_errcode_visit

- Column Name: v_errcode_visit
- Description: Error code for HELIOV_VISIT. Zero is good, nonzero is bad.
- Unit: N/A
- Data Type: Int16


#### velvarflag

- Column Name: velvarflag
- Description: Flag indicating the significant variation of the heliocentric velocity from visit to visit. A value of 1 means the variation is beyond 3-sigma between at least two of the visits. A value of 0 means all variations between pairs of visits are less than 3-sigma.
- Unit: N/A
- Data Type: Int16


#### dv_maxsig

- Column Name: dv_maxsig
- Description: Maximum significance in velocity variation between pairs of visits.
- Unit: N/A
- Data Type: Float32


#### mjdqual

- Column Name: mjdqual
- Description: Spectral quality bitmask (MASTAR_QUAL).
- Unit: N/A
- Data Type: Int32


#### bprperr_sp

- Column Name: bprperr_sp
- Description: Uncertainty in the synthetic Bp-Rp color derived from the spectra.
- Unit: N/A
- Data Type: Float32


#### nexp_used

- Column Name: nexp_used
- Description: Number of exposures used in constructing the visit spectrum.
- Unit: N/A
- Data Type: Int16


#### s2n

- Column Name: s2n
- Description: Median signal-to-noise ratio per pixel of this visit spectrum.
- Unit: N/A
- Data Type: Float32


#### s2n10

- Column Name: s2n10
- Description: Top 10-th percentile signal-to-noise ratio per pixel of this visit spectrum.
- Unit: N/A
- Data Type: Float32


#### badpixfrac

- Column Name: badpixfrac
- Description: Fraction of bad pixels (those with inverse variance being zero.)
- Unit: N/A
- Data Type: Float32


#### coord_source

- Column Name: coord_source
- Description: Source of astrometry.
- Unit: N/A
- Data Type: String


#### photocat

- Column Name: photocat
- Description: Source of photometry.
- Unit: N/A
- Data Type: String


#### pk

- Column Name: pk
- Description: Primary key. Sequential identifier for this table.
- Unit: N/A
- Data Type: Int64
