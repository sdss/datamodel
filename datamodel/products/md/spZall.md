## Basic Information
The spZall files contain the spectroscopic classifications and redshifts for all fits done in the spectro-1d step of the BOSS idlspec1d pipeline.

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/[FIELD]/[RUN1D]/spZall-[FIELD]-[MJD].fits

### Releases
DR14, DR15, DR16, DR17, DR18, IPL3, DR19

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
64 MB

### File Type
FITS

### Generated by Product
idlspec2d - spreduce1d_empca.pro

### Is a VAC
False

### Data Level
0.0.0
