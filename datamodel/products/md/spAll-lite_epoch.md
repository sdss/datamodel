## Basic Information
This file contains a summary of the BOSS idlspec2d and spectro-1d pipeline outputs every field-mjd-target for field epoch coadds

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/epoch/spAll-lite-[RUN2D].fits.gz

### Releases
WORK

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
907 MB

### File Type
FITS

### Generated by Product
idlspec2d

### Is a VAC
False

### HDUS List for release WORK
  - [HDU0: PRIMARY](#hdu0-primary)
  - [HDU1: SPALL](#hdu1-spall)
