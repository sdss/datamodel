## Basic Information
This file contains information from the proc-gimg raw guider image files in the form of structures for SDSS Plate observations. For days where there are no observations this file is still created with the header and the structure definition only.

### Naming Convention
$SPECLOG_DIR/[MJD]/guiderMon-[MJD].par

### Releases
IPL4

### Enviroments
SPECLOG_DIR

### Approximate Size
940 bytes

### File Type
PAR

### Generated by Product
idlspec2d - guidermonfile.pro

### Is a VAC
False

### Data Level
0.3.1

### PAR List for release IPL4
  - [GUIDEOBJ](#GUIDEOBJ)

## Notes
Some of these files may not be readable by pydl.yanny tools, and can only be read in idl using idlutils. The issue comes from missing brackets 
around array definitions in the yanny tables.
