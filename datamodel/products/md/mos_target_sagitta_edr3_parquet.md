## Basic Information
Catalog of pre-main-sequence stars derived from Gaia EDR3 and 2MASS from McBride et al. (2021), their Table 5.

For complete details, see the original paper: <a href="https://iopscience.iop.org/article/10.3847/1538-3881/ac2432">https://iopscience.iop.org/article/10.3847/1538-3881/ac2432</a>

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_sagitta_edr3.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

## Example DATAFRAME List

### Columns


#### source_id

- Column Name: source_id
- Description: Unique identifier in the Gaia (e)DR3 source catalogue
- Unit: N/A
- Data Type: Int64


#### l

- Column Name: l
- Description: Galactic longitude
- Unit: deg
- Data Type: Float64


#### b

- Column Name: b
- Description: Galactic latitude
- Unit: deg
- Data Type: Float64


#### age

- Column Name: age
- Description: Age
- Unit: log yrs
- Data Type: Float32


#### age_std

- Column Name: age_std
- Description: Error on age
- Unit: log yrs
- Data Type: Float64


#### pms

- Column Name: pms
- Description: Pre-main-sequence probability
- Unit: N/A
- Data Type: Float32


#### pms_std

- Column Name: pms_std
- Description: Error on pre-main-sequence probability
- Unit: N/A
- Data Type: Float64


#### av

- Column Name: av
- Description: Predicted extinction along the line of sight (positional only)
- Unit: mag
- Data Type: Float32


#### av_std

- Column Name: av_std
- Description: Uncertainty in the predicted extinction
- Unit: mag
- Data Type: Float64
