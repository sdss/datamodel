## Basic Information
apDark frames contain the measured dark current for each detector. Dark frames exist in a data cube representing the measure dark at each successive readout in an up-the-ramp data collection. Each plane in apDark represents the average dark current at the corresponding readout. The multiextension FITS file contains:
HDU0: master header
HDU1: dark current image (ADU) [FLOAT]: 2048x2048xmaxread
HDU2: flag mask [INT*2]
HDU3: chi2 image [FLOAT]
