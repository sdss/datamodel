## Basic Information
This file contains raw spectro data from the BOSS/eBOSS/BHM spectro CCDs. Simultaneous frames in different chips all have the same frame number. Which frames are what type of observation is indicated by the FLAVOR keyword in each header. HDU 0 contains the raw counts as integers (4352x4224). For SDSS-I raw spectro data, please look at SDSS-I,-II sdR files:https://data.sdss.org/datamodel/files/RAWDATA_DIR/MJD/sdR.html.

### Naming Convention
sdR-[br][12]-[FRAME].fit, indicating which blue or red spectrograph (1 or 2), and the frame number.
