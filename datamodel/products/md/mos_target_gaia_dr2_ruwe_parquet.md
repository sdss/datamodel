## Basic Information
Table from Gaia DR2 containing the Renormalised Unit Weight Error (RUWE) associated to each source in gaia_source.

See <a href="https://gea.esac.esa.int/archive/documentation/GDR2/Gaia_archive/chap_datamodel/sec_dm_main_tables/ssec_dm_ruwe.html">https://gea.esac.esa.int/archive/documentation/GDR2/Gaia_archive/chap_datamodel/sec_dm_main_tables/ssec_dm_ruwe.html</a> for more detailed descriptions of the columns
