## Basic Information
This table contains estimates of the "fidelity" of Gaia eDR3 astrometric solutions, a measure of the likelihood the eDR3 solution is physical rather than spurious obtained using a neural network trained on a small, hand-selected sample.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_gedr3spur_main.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

## Example DATAFRAME List

### Columns


#### source_id

- Column Name: source_id
- Description: Gaia DR3 unique source identifier. Matches the source_id column in the dr20_gaia_dr3_source table.
- Unit: N/A
- Data Type: Int64


#### fidelity_v2

- Column Name: fidelity_v2
- Description: A probability that eDR3 has a good astrometric solution for this source, with values between 0 (meaning likely spurious solution) and 1 (meaning likely good solution). This is the published probability estimate.
- Unit: N/A
- Data Type: Float32


#### theta_arcsec_worst_source

- Column Name: theta_arcsec_worst_source
- Description: Distance to the eDR3 source within 30 arcsec of the object for which delta_g-theta is maximal. See norm_dg for details.
- Unit: arcsec
- Data Type: Float32


#### norm_dg

- Column Name: norm_dg
- Description: This is a heuristic measure for contamination by bright stars in the neighbourhood. It is computed as delta_g-theta, where theta is the distance to another Gaia eDR3 object in arcsec (reported in theta_arcsec_worst_source), and delta_g is the magnitude difference in mag. This column gives the maximum of the values for all eDR3 sources within 30 arcsecs of the object.
- Unit: N/A
- Data Type: Float32


#### dist_nearest_neighbor_at_least_m2_brighter

- Column Name: dist_nearest_neighbor_at_least_m2_brighter
- Description: Distance to the nearest neighbour in gaia_source at least 2 m fainter than this source.
- Unit: arcsec
- Data Type: Float32


#### dist_nearest_neighbor_at_least_0_brighter

- Column Name: dist_nearest_neighbor_at_least_0_brighter
- Description: Distance to the nearest neighbour in gaia_source at least as bright as this source.
- Unit: arcsec
- Data Type: Float32


#### dist_nearest_neighbor_at_least_2_brighter

- Column Name: dist_nearest_neighbor_at_least_2_brighter
- Description: Distance to the nearest neighbour in gaia_source at least 2 m brighter than this source.
- Unit: arcsec
- Data Type: Float32


#### dist_nearest_neighbor_at_least_4_brighter

- Column Name: dist_nearest_neighbor_at_least_4_brighter
- Description: Distance to the nearest neighbour in gaia_source at least 4 m brighter than this source.
- Unit: arcsec
- Data Type: Float32


#### dist_nearest_neighbor_at_least_6_brighter

- Column Name: dist_nearest_neighbor_at_least_6_brighter
- Description: Distance to the nearest neighbour in gaia_source at least 6 m brighter than this source.
- Unit: arcsec
- Data Type: Float32


#### dist_nearest_neighbor_at_least_10_brighter

- Column Name: dist_nearest_neighbor_at_least_10_brighter
- Description: Distance to the nearest neighbour in gaia_source at least 10 m brighter than this source.
- Unit: arcsec
- Data Type: Float32


#### fidelity_v1

- Column Name: fidelity_v1
- Description: A probablity that eDR3 has a good astrometric solution for this source, with values between 0 (meaning likely spurious solution) and 1 (meaning likely good solution). This comes from a first version of the estimator that was reviewed based on an astro-ph paper.
- Unit: N/A
- Data Type: Float32
