## Basic Information
The spTraceTab file contains the trace table data for a given MJD and Observatory, including information about nightly calibration exposures. It is used to build the traces for the spectrograph data reduction process using an inital set of arc and flat exposures, and correlating theses traces with arc frames taken for each science field during the night

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/trace/[MJD]/spTraceTab-[BR][ID]-[FRAME].fits
