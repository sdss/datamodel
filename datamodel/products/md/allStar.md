## Basic Information
The allStar-VERS-TELESCOPE.fits file contains all final data product and summary information for every observed star in the data release.

### Naming Convention
<code>allStar-VERS-TELESCOPE.fits</code>

### Releases
DR10, DR12, DR13, DR14, DR15, DR16, DR17, DR19

### Enviroments
APOGEE_REDUX, APOGEE_ASPCAP

### Approximate Size
148 MB

### File Type
FITS

### Generated by Product
apogeedrp.py

### Is a VAC
False

### Data Level
2.2.0

### HDUS List for release DR19
  - [HDU0: PRIMARY](#hdu0-primary)
  - [HDU1: APOGEE reduction star-level summary table](#hdu1-apogee reduction star-level summary table)
