## Basic Information
The parameters for the metrics that describe a given Design Mode, where a Design Mode constrains the assignments on a design.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_design_mode.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

### Releases
DR20

### Enviroments
MOS_TARGET

### Approximate Size
39 KB

### File Type
PARQUET

### Generated by Product
sdss5db targeting database.

### Is a VAC
False

### Data Level
2.3.3

### DATAFRAME List for release DR20
