## Basic Information
List of APOGEE DR16 stars for RV follow-up.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_sdss_apogeeallstarmerge_r13.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

### Releases
DR20

### Enviroments
MOS_TARGET

### Approximate Size
96 MB

### File Type
PARQUET

### Generated by Product
sdss5db targeting database.

### Is a VAC
False

### Data Level
2.3.3

### DATAFRAME List for release DR20

## Example DATAFRAME List

### Columns


#### apogee_id

- Column Name: apogee_id
- Description: 2MASS style ID from APOGEE DR16
- Unit: N/A
- Data Type: String


#### nvisits

- Column Name: nvisits
- Description: Number of visits into combined spectra, accross all allStar entries for the star
- Unit: N/A
- Data Type: Int16


#### nentries

- Column Name: nentries
- Description: Number of unique allStar entries for the star
- Unit: N/A
- Data Type: Int32


#### ra

- Column Name: ra
- Description: Right ascension (J2000)
- Unit: degrees
- Data Type: Float64


#### dec

- Column Name: dec
- Description: Declination (J2000)
- Unit: degrees
- Data Type: Float64


#### glon

- Column Name: glon
- Description: Galactic longitude
- Unit: degrees
- Data Type: Float64


#### glat

- Column Name: glat
- Description: Galactic latitude
- Unit: degrees
- Data Type: Float64


#### pmra

- Column Name: pmra
- Description: proper motion in RA
- Unit: mas/yr
- Data Type: Float64


#### pmdec

- Column Name: pmdec
- Description: proper motion in DEC
- Unit: mas/yr
- Data Type: Float64


#### pm_src

- Column Name: pm_src
- Description: source of proper motion (e.g. gaia)
- Unit: N/A
- Data Type: String


#### j

- Column Name: j
- Description: 2MASS J mag [bad=99]
- Unit: mag
- Data Type: Float32


#### j_err

- Column Name: j_err
- Description: Uncertainty in 2MASS J mag
- Unit: mag
- Data Type: Float32


#### h

- Column Name: h
- Description: 2MASS H mag [bad=99]
- Unit: mag
- Data Type: Float32


#### h_err

- Column Name: h_err
- Description: Uncertainty in 2MASS H mag
- Unit: mag
- Data Type: Float32


#### k

- Column Name: k
- Description: 2MASS Ks mag [bad=99]
- Unit: mag
- Data Type: Float32


#### k_err

- Column Name: k_err
- Description: Uncertainty in 2MASS Ks mag
- Unit: mag
- Data Type: Float32


#### ak

- Column Name: ak
- Description: K-band extinction adopted for targetting
- Unit: mag
- Data Type: Float32


#### vhelio_avg

- Column Name: vhelio_avg
- Description: Average radial velocity, weighted by S/N, rederived to reflect all entries
- Unit: km/s
- Data Type: Float32


#### vhelio_err

- Column Name: vhelio_err
- Description: Uncertainty in VHELIO_AVG from the S/N-weighted individual RVs, rederived to reflect all entries
- Unit: km/s
- Data Type: Float32


#### vscatter

- Column Name: vscatter
- Description: Scatter of individual visit RVs around average, rederived to reflect all entries
- Unit: km/s
- Data Type: Float32


#### sig_rvvar

- Column Name: sig_rvvar
- Description: Measure of the significance of the star's RV variability (see Troup, et. al 2016)
- Unit: N/A
- Data Type: Float32


#### baseline

- Column Name: baseline
- Description: Temporal baseline of the observations (JD_last-JD-first)
- Unit: days
- Data Type: Float32


#### mean_fiber

- Column Name: mean_fiber
- Description: The mean fiberID of all of the star's visits
- Unit: N/A
- Data Type: Float32


#### sig_fiber

- Column Name: sig_fiber
- Description: The standard deviation of the fiberID of all of the star's visits
- Unit: N/A
- Data Type: Float32


#### apstar_ids

- Column Name: apstar_ids
- Description: List of APSTAR_ID from each of the star's allStar entries
- Unit: N/A
- Data Type: String


#### visits

- Column Name: visits
- Description: List of VISIT_ID from eeach of the star's allVisit entries
- Unit: N/A
- Data Type: String


#### fields

- Column Name: fields
- Description: List of FIELD flags from each of the star's allStar entries
- Unit: N/A
- Data Type: String


#### surveys

- Column Name: surveys
- Description: List of SURVEY flags from each of the star's allStar entries
- Unit: N/A
- Data Type: String


#### telescopes

- Column Name: telescopes
- Description: List of TELESCOPE flags from each of the star's allStar entries
- Unit: N/A
- Data Type: String


#### targflags

- Column Name: targflags
- Description: Verbose/text form of TARGFLAG combined from all entries
- Unit: N/A
- Data Type: String


#### starflags

- Column Name: starflags
- Description: Verbose/text form of STARFLAG combined from all entries
- Unit: N/A
- Data Type: String


#### aspcapflags

- Column Name: aspcapflags
- Description: Verbose/text form of ASPCAPFLAG combined from all entries
- Unit: N/A
- Data Type: String


#### teff

- Column Name: teff
- Description: Teff from ASPCAP analysis of combined spectra (from PARAM) averaged across entries
- Unit: K
- Data Type: Float32


#### teff_err

- Column Name: teff_err
- Description: Teff uncertainty (from PARAM_COV)
- Unit: K
- Data Type: Float32


#### logg

- Column Name: logg
- Description: log g from ASPCAP analysis of combined spectrum (from PARAM) averaged across entries
- Unit: dex
- Data Type: Float32


#### logg_err

- Column Name: logg_err
- Description: log g uncertainty (from PARAM_COV)
- Unit: dex
- Data Type: Float32


#### feh

- Column Name: feh
- Description: metallicity from ASPCAP analysis of combined spectrum (from PARAM) averaged across entries
- Unit: dex
- Data Type: Float32


#### feh_err

- Column Name: feh_err
- Description: feh uncertainty (from PARAM_COV)
- Unit: dex
- Data Type: Float32


#### startype

- Column Name: startype
- Description: Best guess of star's evolutionary state based on stellar parameters or external catalogs (RC=red clump, RG=red giant, SG = subgiant, MS=dwarf, PMS=pre-main sequence)
- Unit: N/A
- Data Type: String


#### vjitter

- Column Name: vjitter
- Description: Atmospheric RV jitter derived from log g using the relationship derived in Hekker, et. al 2008.
- Unit: km/s
- Data Type: Float32


#### dist

- Column Name: dist
- Description: Derived or given distance to the star
- Unit: pc
- Data Type: Float32


#### dist_err

- Column Name: dist_err
- Description: Uncertainty of the distance
- Unit: pc
- Data Type: Float32


#### dist_src

- Column Name: dist_src
- Description: Source of the star's distance measurement (e.g. gaia=derived from gaia parallax)
- Unit: N/A
- Data Type: String


#### mstar

- Column Name: mstar
- Description: Derived or given stellar mass (Solar Mass)
- Unit: N/A
- Data Type: Float32


#### mstar_err

- Column Name: mstar_err
- Description: Uncertainty of the Derived stellar mass
- Unit: Solar Mass
- Data Type: Float32


#### rstar

- Column Name: rstar
- Description: Derived stellar radius
- Unit: Solar Radius
- Data Type: Float32


#### rstar_err

- Column Name: rstar_err
- Description: Uncertainty of the Derived stellar radius
- Unit: Solar Radius
- Data Type: Float32


#### mstar_src

- Column Name: mstar_src
- Description: Source of the star's Mass (if not derived)
- Unit: N/A
- Data Type: String


#### designation

- Column Name: designation
- Description: identifier
- Unit: N/A
- Data Type: String
