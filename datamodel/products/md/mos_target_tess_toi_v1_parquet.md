## Basic Information
This catalog contains targets that received the 2 minute cadence during the TESS Mission, are TESS Objects of Interest (TOI) or Community TESS Objects of Interest (CTOI).

The contents of this catalog were derived from the MIT TESS website (<a href="https://tess.mit.edu/public/target_lists/target_lists.html)">https://tess.mit.edu/public/target_lists/target_lists.html)</a> and the ExoFOP website <a href="https://exofop.ipac.caltech.edu/tess/index.php">https://exofop.ipac.caltech.edu/tess/index.php</a>. These targets were updated on 2022-08-05.

### Naming Convention

## Example DATAFRAME List

### Columns


#### ticid

- Column Name: ticid
- Description: TESS Input Catalog (TIC) ID
- Unit: N/A
- Data Type: Int64


#### target_type

- Column Name: target_type
- Description: Type of target. Options: 2min = 2 minute cadence, exo_TOI = TOI, and exo_CTOI = CTOI
- Unit: N/A
- Data Type: String


#### toi

- Column Name: toi
- Description: TESS Object of Interest ID
- Unit: N/A
- Data Type: String


#### tess_disposition

- Column Name: tess_disposition
- Description: TESS Team Dispositon Options: EB = Eclipsing Binary, KP = Known Planet, PC = Planet Candidate, O = Other, or blank
- Unit: N/A
- Data Type: String


#### tfopwg_disposition

- Column Name: tfopwg_disposition
- Description: TESS Follow-up Observing Program (TFOP) Working Group Disposition Options: APC = Ambiguous Planet Candidate, CP = Confirmed Planet, FA = False Alarm, FP = False Positive, KP = Known Planet, PC = Planet Candidate or blank
- Unit: N/A
- Data Type: String


#### ctoi

- Column Name: ctoi
- Description: Community Tareget of Interest ID
- Unit: N/A
- Data Type: String


#### user_disposition

- Column Name: user_disposition
- Description: Initial Community User Disposition
- Unit: N/A
- Data Type: String


#### num_sectors

- Column Name: num_sectors
- Description: The total number of sectors that this object was observed on as of the catalog date
- Unit: N/A
- Data Type: Float64


#### pkey

- Column Name: pkey
- Description: Primary Key
- Unit: N/A
- Data Type: Int64
