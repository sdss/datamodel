## Basic Information
The GALAH DR3 main catalogue. See <a href="https://www.galah-survey.org/dr3/table_schema/">https://www.galah-survey.org/dr3/table_schema/</a>

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_galah_dr3.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

### Releases

### Enviroments

223 MB

### File Type
PARQUET

### Generated by Product

### Is a VAC
