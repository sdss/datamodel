## Basic Information
CSC2 X-ray sources.

This catalogue contains CSC2 X-ray sources that have been matched separately to PanSTARRS-1, Gaia DR2, and 2MASS catalogs, using the programs NWAY (Johannes Buchner; see: Salvato 2018, MN, 473, 4937) and Xmatch (Arnold Rots; see: <a href="https://cxc.cfa.harvard.edu/csc/csc_crossmatches.html)">https://cxc.cfa.harvard.edu/csc/csc_crossmatches.html)</a>. Both are based on the Bayesian spatial cross-matching algorithm developed by Budavari & Szalay (2008, ApJ 679, 301), but Xmatch has the added capability of taking source extent and/or PSF into account. Created March 2021 by Paul Green, Dong-Woo Kim, Arnold Rots and the CXC CatSci group.

## Example DATAFRAME List

### Columns


#### cxoid

- Column Name: cxoid
- Description: CSC2 Chandra ID
- Unit: N/A
- Data Type: String


#### xra

- Column Name: xra
- Description: RA of X-ray source from CSC2
- Unit: deg
- Data Type: Float64


#### xdec

- Column Name: xdec
- Description: Dec of X-ray source from CSC2
- Unit: deg
- Data Type: Float64


#### pri

- Column Name: pri
- Description: Priority (based on X-ray S/N)
- Unit: N/A
- Data Type: Int16


#### ocat

- Column Name: ocat
- Description: optical catalog code (P for PS1, G for Gaia)
- Unit: N/A
- Data Type: String


#### oid

- Column Name: oid
- Description: object ID from optical catalog
- Unit: N/A
- Data Type: Int64


#### ora

- Unit: deg
- Data Type: Float64


#### odec

- Column Name: odec

- Unit: deg
- Data Type: Float64


#### omag

- Column Name: omag

- Unit: mag
- Data Type: Float32


#### omatchtype

- Column Name: omatchtype
- Description: X-ray/optical match type (1-4 definite/likely/multiple opt, multiple X)
- Unit: N/A
- Data Type: Int16


#### irid

- Column Name: irid

- Unit: N/A
- Data Type: String
