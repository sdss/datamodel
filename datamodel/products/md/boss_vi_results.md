## Basic Information
Catalog of redshifts/classifications/and confidence flags derived from visual inspections of BOSS allepoch spectra, supplemented with ML predictions of pipeline redshift reliability.

### Naming Convention
$BHM_VI/[V_VAC]/boss_vi_results-[V_VAC].fits

### Releases
DR20

### Enviroments
BHM_VI

### Approximate Size
28 MB

### File Type
FITS

### Generated by Product


### Is a VAC
False

### Data Level
2.3.3
