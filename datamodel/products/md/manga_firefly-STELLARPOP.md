## Basic Information
manga-firefly-v2_4_3-GLOBALPROP.fits is one of the MaNGA FIREFLY catalogues for the the 15th data release of the Sloan Digital Sky Survey. The file contains analysis for 4675 galaxies from the MaNGA survey, providing global properties such as gradients and absorption. For DR15, the version of FIREFLY used was FIREFLY_VER=v1_1_0, and this was run on MANGADRP_VER=v2_4_3 reduced data.  The catalogue is available in FITS format at the following link: <dd> <dd> <a href="https://data.sdss.org/sas/dr15/manga/spectro/firefly/v1_1_2/manga_firefly-v2_4_3-STELLARPOP.fits" target="_blank">https://data.sdss.org/sas/dr15/manga/spectro/firefly/v1_1_2/manga_firefly-v2_4_3-STELLARPOP.fits</a> </dd> <dt>Naming Convention</dt> <dd id="filename"><code> filename = manga_firefly-MANGADRP_VER-STELLARPOP.fits where MANGADRP_VER=v2_4_3 for DR15 </code>.</dd> <dt>Approximate Size</dt> <dd id="filesize">2 GB</dd> <dt>File Type</dt> <dd id="filetype">FITS</dd> <dt>Read by Products</dt> <dd>firefly</dd> <dt>Written by Products</dt> <dd>firefly</dd> <dt>Sections</dt> <dd><p>This should contain internal links to parts of the file (if any).</p> <ul> <li><a href="#hdu0">HDU0</a>: The Primary Header</li> <li><a href="#hdu1">HDU1</a>: A Data Table</li> <li><a href="#example">Examples</a></li> </ul> </dd> </dd>
