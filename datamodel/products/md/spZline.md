## Basic Information
This file contains the emission-line fits from Spectro-1D.

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/fields/[FIELDGRP]/[FIELD]/[RUN1D]/spZline-[FIELD]-[MJD].fits

### Releases
DR9, DR10, DR12, DR11, DR13, DR14, DR15, DR16, DR17, DR18, IPL3, DR19, IPL4, DR20

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
10 MB

### File Type
FITS

### Generated by Product
idlspec2d - spreduce1d_empca.pro

### Is a VAC
False

### Data Level
1.3
