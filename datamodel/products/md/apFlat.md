## Basic Information
apFlat files contain calibration data about the relative response of the different pixels in each detector. They are constructed from multiple frames of a continuum lamp, taken at different collimator dither positions. The multiextension FITS file contains:
HDU0: master header
HDU1: flat field image (ADU) [FLOAT]: 2048x2048
HDU2: flag mask [INT*2]
HDU3: smoothed flat image [FLOAT]


### Naming Convention
$APOGEE_REDUX/[APRED]/cal/[INSTRUMENT]/flatcorr/apFlat-[CHIP]-[NUM8].fits
