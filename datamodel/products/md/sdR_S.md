## Basic Information
This file contains raw spectro data from the BHM+MWM spectro CCDs from LCO. Simultaneous frames in different chips all have the same frame number. Which frames are what type of observation is indicated by the FLAVOR keyword in each header. HDU 0 contains the raw counts as integers (4352x4224).

### Naming Convention
sdR-[br][id]-[FRAME].fit, indicating which blue or red spectrograph (1 or 2), and the frame number.

### Releases
IPL4, DR20

### Enviroments
BOSS_SPECTRO_DATA_S

### Approximate Size
11 MB

### File Type
FIT

### Generated by Product
OPS

### Is a VAC
False

### Data Level
0.1.2

### HDUS List for release DR20
  - [HDU0: PRIMARY](#hdu0-primary)
