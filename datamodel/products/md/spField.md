## Basic Information
The spField files contain the single mjd combined spectra for all exposures of a given field mjd. This is the updated SDSSV version of the spPlate files and analogous to the spField files

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/fields/[FIELDGRP]/[FIELD]/spField-[FIELD]-[MJD].fits

### Releases
DR18, IPL3, DR19, IPL4, DR20

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
63 MB

### File Type
FITS

### Generated by Product
idlspec2d

### Is a VAC
False

### Data Level
2.2.3
