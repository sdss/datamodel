## Basic Information
Bayesian distances from Gaia DR2 parameters from Bailer-Jones et al. 2018.

For complete details, see the original paper: <a href="https://iopscience.iop.org/article/10.3847/1538-3881/aacb21/pdf">https://iopscience.iop.org/article/10.3847/1538-3881/aacb21/pdf</a>

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_geometric_distances_gaia_dr2.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

## Example DATAFRAME List

### Columns


#### source_id

- Column Name: source_id
- Description: Unique Gaia DR2 source identifier
- Unit: N/A
- Data Type: Int64


#### r_est

- Column Name: r_est
- Description: Estimated distance
- Unit: pc
- Data Type: Float32


#### r_lo

- Column Name: r_lo
- Description: Lower bound on the confidence interval of the estimated distance
- Unit: pc
- Data Type: Float32


#### r_hi

- Column Name: r_hi
- Description: Upper bound on the confidence interval of the estimated distance
- Unit: pc
- Data Type: Float32


#### r_len

- Column Name: r_len
- Description: Length scale used in the prior for the distance estimation
- Unit: pc
- Data Type: Float32


#### result_flag

- Column Name: result_flag
- Description: Result flag
- Unit: N/A
- Data Type: String


#### modality_flag

- Column Name: modality_flag
- Description: Number of modes in the posterior
- Unit: N/A
- Data Type: Int16
