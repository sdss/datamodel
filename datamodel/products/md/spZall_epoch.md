## Basic Information
The spZall files contain the spectroscopic classifications and redshifts for all fits done in the spectro-1d step of the BOSS idlspec1d pipeline for field epoch coadds

### Naming Convention
$BOSS_SPECTRO_REDUX/[RUN2D]/fields/[FIELDGRP]/[FIELD]/epoch/[RUN1D]/spZall-[FIELD]-[MJD].fits

### Releases
IPL3, DR19, IPL4, DR20

### Enviroments
BOSS_SPECTRO_REDUX

### Approximate Size
66 MB

### File Type
FITS

### Generated by Product
idlspec2d

### Is a VAC
False

### Data Level
2.2.3

### HDUS List for release DR20
  - [HDU0: PRIMARY](#hdu0-primary)
  - [HDU1: ZALL](#hdu1-zall)
