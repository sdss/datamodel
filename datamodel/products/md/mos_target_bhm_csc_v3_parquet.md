## Example DATAFRAME List

### Columns


#### csc21p_id

- Column Name: csc21p_id
- Description: CSC 2.1 source name in the form "2CXO Jhhmmss.s{+|-}ddmmss"
- Unit: N/A
- Data Type: String


#### csc21p_ensemble

- Column Name: csc21p_ensemble
- Description: CSC 2.1 ensemble. An ensemble is a set of overlapping stacks. It represents all the data used to match detections (i.e. to identify sources), and for calculating source properties.
- Unit: N/A
- Data Type: String


#### ra

- Column Name: ra
- Description: Preferred counterpart RA coordinate
- Unit: deg
- Data Type: Float64


#### dec

- Column Name: dec
- Description: Preferred counterpart Dec coordinate
- Unit: deg
- Data Type: Float64


#### best_oir_cat

- Column Name: best_oir_cat
- Description: Best optical counterpart match; 'gdr3', 'lsdr10', 'ps1dr2', or '2mass' in that order of decreasing priority.
- Unit: N/A
- Data Type: String


#### best_mag

- Column Name: best_mag
- Description: Preferred magnitude of the optical counterpart
- Unit: mag
- Data Type: Float32


#### mag_type

- Column Name: mag_type
- Description: Source of best_mag; GAIA_G, LS_g,LS_r, LS_z, PS_gMeanPSFMag, PS_rMeanPSFMag, PS_iMeanPSFMag, PS_zMeanPSFMag,2MASS_h_m in that order of decreasing priority.
- Unit: N/A
- Data Type: String


#### gaia_dr3_srcid

- Column Name: gaia_dr3_srcid
- Description: Gaia DR3 source ID
- Unit: N/A
- Data Type: Int64


#### ls_dr10_lsid

- Column Name: ls_dr10_lsid
- Description: Legacy Survey DR10 ID
- Unit: N/A
- Data Type: Int64


#### ls_dr10_objid

- Column Name: ls_dr10_objid
- Description: Legacy Survey DR10 object ID
- Unit: N/A
- Data Type: Int32


#### ls_dr10_brickid

- Column Name: ls_dr10_brickid
- Description: Legacy Survey DR10 brick ID
- Unit: N/A
- Data Type: Int32


#### ls_dr10_release

- Column Name: ls_dr10_release
- Description: Legacy Survey DR10 release
- Unit: N/A
- Data Type: Int16


#### ps21p_objid

- Column Name: ps21p_objid
- Description: Pan-STARRS1 DR2 IPP unique object identifier
- Unit: N/A
- Data Type: Int64


#### ps21p_ippobjid

- Column Name: ps21p_ippobjid
- Description: Pan-STARRS1 DR2 IPP internal object identifier
- Unit: N/A
- Data Type: Int64


#### tmass_designation

- Column Name: tmass_designation
- Description: 2MASS designation
- Unit: N/A
- Data Type: String


#### tmass_ra

- Column Name: tmass_ra
- Description: RA from 2MASS
- Unit: deg
- Data Type: Float64


#### tmass_dec

- Column Name: tmass_dec
- Description: Dec from 2MASS
- Unit: deg
- Data Type: Float64


#### tmass_h

- Column Name: tmass_h
- Description: H-band magnitude from 2MASS
- Unit: mag
- Data Type: Float32


#### csc21p_ra

- Column Name: csc21p_ra
- Description: RA of X-ray source
- Unit: N/A
- Data Type: Float64


#### csc21p_dec

- Column Name: csc21p_dec
- Description: DEC of X-ray source
- Unit: N/A
- Data Type: Float64


#### logfx

- Column Name: logfx
- Description: log10 of X-ray flux in xband
- Unit: N/A
- Data Type: Float64


#### xband

- Column Name: xband
- Description: X-ray bandpass for logfx, priority b,m,s,h,w
- Unit: N/A
- Data Type: String


#### extent

- Column Name: extent
- Description: X-ray extent flag
- Unit: N/A
- Data Type: Boolean


#### xsig

- Column Name: xsig
- Description: X-ray significance
- Unit: N/A
- Data Type: Float64


#### xpriority

- Column Name: xpriority
- Description: X-ray priority, calculated from significance
- Unit: N/A
- Data Type: Int32


#### pk

- Column Name: pk
- Description: Primary key identifier for this table row
- Unit: N/A
- Data Type: Int64
