## Basic Information
The Eighth version (v8.0) of the TESS Input Catalogue (<a href="<a href="https://outerspace.stsci.edu/display/TESS/TIC+v8+and+CTL+v8.xx+Data+Release+Notes"></a>)">https://outerspace.stsci.edu/display/TESS/TIC+v8+and+CTL+v8.xx+Data+Release+Notes"></a>)</a>.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_tic_v8.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

### Releases
DR20

### Enviroments
MOS_TARGET

### Approximate Size
15 GB

### File Type
PARQUET

### Generated by Product
sdss5db targeting database.

### Is a VAC
False

### Data Level
2.3.3

### DATAFRAME List for release DR20
- Primary data frame.
