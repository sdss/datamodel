## Basic Information
Cadences define the time between series of exposures (one "epoch") on a field, and the observing conditions, e.g., skybrightness, for each epoch.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_cadence.[FTYPE] with FTYPE='fits' and V_TARG='2.0.0' for DR20.

### Releases
DR18, DR19, DR20

### Enviroments
MOS_TARGET

### Approximate Size
36 KB

### File Type
FITS

### Generated by Product
sdss5db targeting database.

### Is a VAC
False

### Data Level
2.3.3
