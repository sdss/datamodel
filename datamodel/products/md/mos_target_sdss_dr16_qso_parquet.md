## Basic Information
SDSS Data Release 16 Quasar Catalog (Lyke et al. 2020).

For complete details, please see the original paper: <a href="https://ui.adsabs.harvard.edu/abs/2020ApJS">https://ui.adsabs.harvard.edu/abs/2020ApJS</a>..250....8L/abstract.

Description derived from SDSS datamodel: <a href="https://data.sdss.org/datamodel/files/BOSS_QSO/DR16Q/DR16Q_v4.html">https://data.sdss.org/datamodel/files/BOSS_QSO/DR16Q/DR16Q_v4.html</a>

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_sdss_dr16_qso.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.
