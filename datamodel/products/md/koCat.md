## Basic Information
Survey Interface File for Known Object Catalog.
Input catalog of known objects overlapping run being processed.

### Naming Convention
koCat-rrrrrr.fit, where <i>rrrrrr</i> is the imaging run number. <b>NOTE</b> : this naming schema does not support <i>kali</i> .
