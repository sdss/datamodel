## Basic Information
This table stores the targets associated with DR20 target selection cartons. Note that the targets in this table are unique, but a target can be associated with multiple cartons. That many-to-many relationship is encoded in the dr20_carton_to_target table.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_target-[NUM].[FTYPE] with FTYPE='fits', V_TARG='2.0.0', and NUM=1..94 for DR20.

### Releases
DR18, DR19, DR20

### Enviroments
MOS_TARGET

### Approximate Size
91 MB

### File Type
FITS

### Generated by Product
sdss5db targeting database.

### Is a VAC
False

### Data Level
