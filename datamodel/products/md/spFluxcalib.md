## Basic Information
This file contains the calibration vectors for a given frame.
<pre>
  HDU #0: Calibration image
  HDU #1: Trace-set used to construct the former
  HDU #2: Structure with info on each standard star  
</pre>

### Naming Convention
spFluxcalib-[c][n]-eeeeeeee.fits.gz
Where [c] indicates either a blue or red spectrograph, [n] indicates the spectrograph number (1 or 2) and eeeeeeee indicates the exposure number.

### Releases
DR9, DR10, DR12, DR11, DR13, DR14, DR15, DR16, DR17

### Enviroments

### Approximate Size
