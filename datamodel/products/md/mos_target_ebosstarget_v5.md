## Basic Information
A catalog of targets, selected from SDSS+WISE imaging that were considered for observation in the

### Naming Convention
$MOS_TARGET/[V_TARG]/mos_target_ebosstarget_v5-[NUM].fits, where V_TARG=1.0.1 for WORK; and NUM = 1..1 to partition the file into parts

### Releases
WORK

### Enviroments
MOS_TARGET

### Approximate Size
555 MB

### File Type
FITS

### Generated by Product
sdss5db> targetdb, operations database server

### Is a VAC
False
