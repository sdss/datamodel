## Basic Information
Reverberation mapping (RM) targets for SDSS-V.

This table enables small modifications to be made to the set of reverberation mapping (RM) targets selected for observation in SDSS-V. The dr20_bhm_rm_tweaks table allows identification of confirmed QSOs which were observed in plate mode that should be preferentially targeted in forward-looking FPS mode observations, and reject candidates which the plate-mode observations have revealed to be unsuitable for continued RM studies.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_bhm_rm_tweaks.[FTYPE] with FTYPE='parquet' and V_TARG='2.0.0' for DR20.

#### rm_suitability

- Column Name: rm_suitability
- Description: Flag indicating if this target is well suited to continued RM observation.
