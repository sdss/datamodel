## Basic Information
The parameters for the metrics that descirbe a given Design Mode, where a Design Mode constrains the assignments on a design.

### Naming Convention
$MOS_TARGET/[V_TARG]/[FTYPE]/mos_design_mode.[FTYPE] with FTYPE='fits' and V_TARG='2.0.0' for DR20.

### Releases
DR19, DR20

### Enviroments
MOS_TARGET

### Approximate Size
30 KB

### File Type
FITS

### Generated by Product
sdss5db> targetdb, operations database server

### Is a VAC
False

### Data Level
2.3.3

### HDUS List for release DR20
  - [HDU0: PRIMARY](#hdu0-primary)
  - [HDU1](#hdu1)
