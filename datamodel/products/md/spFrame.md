## Basic Information
spFrame contains the non-calibrated spectra from the red- or blue-half
of a single exposure.  It is structurally the same as spCFrame which
contains the calibrated spectra, except for the wavelength solution in
HDU 3:  spFrame uses a compact trace-set structure format while spCFrame
uses an evaluated image for the log10(wavelength) at each pixel for each
fiber.
